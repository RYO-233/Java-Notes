## 常用的 DOS 命令

| 命令                     | 说明                       | 示例                   |
| ------------------------ | -------------------------- | ---------------------- |
| ==dir==                  | 查看当前目录有什么内容     | dir d:\abc2\test200    |
| ==cd==(change directory) | 切换到其他盘下             | cd /D c: -> 切换到 C盘 |
| ==cd \\==                | 切换到根目录               |                        |
| ==tree==                 | 查看指定目录下所有子级目录 |                        |
| ==cls==                  | 清屏                       |                        |
| ==exit==                 | 退出 DOS                   |                        |
| md                       | 创建目录                   |                        |
| rd                       | 删除目录                   |                        |
| copy                     | 拷贝文件                   |                        |
| del                      | 删除文件                   |                        |
| echo                     | 输入内容到文件             |                        |
| type，move               | 剪切                       |                        |
